## 🤔 What happens when a village in rural India has no doctor at night?

People guess. People panic. People make wrong decisions.

**700 million rural Indians** have limited or no access to quality healthcare. When someone in a village near Jhansi gets chest pain at midnight — there is no doctor to call, no clinic nearby, and no reliable information available.

**MedSathi changes that.**

> 💡 *"Built by someone from Jhansi — for people who look like my neighbours."*

---

## 🌟 What is MedSathi?

MedSathi is an **AI-powered health assistant** built specifically for rural India. Users describe their symptoms in **Hindi or English** and instantly receive:

- Simple, jargon-free health guidance
- Home remedies and self-care steps  
- Clear warning signs that need immediate doctor attention
- **Emergency alerts** for life-threatening symptoms like chest pain

## 🏗️ How It Works
User describes symptoms (Hindi or English)
↓
MedSathi detects language automatically
↓
Gemma 3 (4B) processes with medical system prompt
↓
Emergency keywords trigger 🚨 ALERT system
↓
Word-by-word typing animation delivers response
↓
Always ends with: "Please visit a doctor"
---

## 🚨 Emergency Detection System

MedSathi automatically scans every AI response for emergency keywords:

- `EMERGENCY` → triggers red pulsing alert box
- `AMBULANCE` → shows emergency numbers (108, 104)  
- `आपातकाल` → Hindi emergency trigger

> This feature exists because in rural India, people often don't know when something is life-threatening.

```bash

git clone https://github.com/DeveshShukla23/MedSathi.git
cd MedSathi

# Install dependencies
pip install streamlit google-genai python-dotenv

# Add your API key
echo "GOOGLE_API_KEY=your_key_here" > .env

# Run the app
streamlit run app.py
```
