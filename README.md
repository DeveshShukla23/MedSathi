## 🌍 The Problem — 700 Million People Without Healthcare

> It is 2 AM in a village near Jhansi, Uttar Pradesh.
> A 60-year-old man has chest pain and difficulty breathing.
> The nearest hospital is 47 kilometres away.
> There is no doctor to call. No reliable information available.
> His family guesses. They wait. They hope.

**This is not a rare situation. This is everyday life for 700 million rural Indians.**

- 🏥 **6 lakh villages** in India have no primary health centre
- 👨‍⚕️ **1 doctor per 1,700 people** in rural areas vs WHO recommendation of 1 per 1,000
- 📵 **Limited internet** in remote areas makes online search unreliable
- 🌐 **Language barrier** — most health information is in English only

**MedSathi was built to solve this — by someone from Jhansi, for people who look like his neighbours.**

---

## 🏥 What is MedSathi?

**MedSathi** (meaning *Health Friend* in Hindi) is an AI-powered health assistant built specifically for rural India using **Google Gemma 3**.

Users describe symptoms in **Hindi or English** and instantly receive:

```
✅  Simple, jargon-free health guidance
✅  Possible causes explained in plain language
✅  Home remedies and self-care steps
✅  Clear warning signs requiring immediate attention
✅  Emergency alerts with helpline numbers (108, 104)
```

> *"Not a doctor. Not a replacement. A knowledgeable friend who is always available."*

## 🤖 How Gemma 3 Powers MedSathi

MedSathi uses **`gemma-3-4b-it`** — Google's instruction-tuned Gemma 3 model — with a carefully engineered system prompt that:

- Forces **simple, non-technical language** appropriate for rural users
- Enforces a **structured response format** (Causes → Home Care → Warning Signs → Doctor Visit)
- Detects **emergency symptoms** and triggers immediate alert protocols
- Switches **language mode** based on user's selection (Hindi/English)
- **Never gives a confirmed diagnosis** — always recommends professional consultation

```python
# Gemma 3 powers MedSathi's intelligence
response = client.models.generate_content(
    model="gemma-3-4b-it",
    contents=full_prompt  # System prompt + user symptoms
)
```

## 🚨 Emergency Detection System

MedSathi automatically scans every Gemma response for emergency keywords:

```
English triggers  →  "EMERGENCY", "AMBULANCE", "IMMEDIATELY"
Hindi triggers    →  "आपातकाल", "तुरंत", "अस्पताल"
```

When triggered:
- 🔴 **Pulsing red alert box** appears instantly
- 📞 **Emergency numbers** displayed: Ambulance 108 | Health 104 | Police 100
- ⚠️ **Bold warning** to go to hospital immediately

> This feature exists because in rural India, people often cannot tell when a situation is life-threatening.

## 🏃 Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/DeveshShukla23/MedSathi.git
cd MedSathi

# 2. Install dependencies
pip install streamlit google-genai python-dotenv

# 3. Create .env file with your API key
echo "GOOGLE_API_KEY=your_google_ai_studio_key" > .env

# 4. Run the app
streamlit run app.py

# 5. Open browser at http://localhost:8501
```
